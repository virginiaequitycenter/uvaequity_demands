INTERACTIVE TIMELINE

We took 50 years of documents, read them, separated them into specific calls to action, and tagged each action item with one or more of 13 key themes. We present those documents here in a timeline format for you to explore within their contexts.

Use the Arrow keys to progress forward and backward through time.


Each group of lines represents ten distinct demands written in a document.


Hover over a line to see the written text.

Click for the full document.

Highlighting means a demand belongs to the theme noted in the top left. We start by highlighting calls for Minority Faculty / Staff Recruitment, but you can use the dropdown menu to change which theme you want to see.
